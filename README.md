# Login
El login les da autenticacion ha ambos roles (admin y medidor) pero cada uno tendra sus propias funcionalidades y necesidades. Ademas de validar sus credenciales para ver si estan registradas o correctas.

# Registro
El registro esta en el mismo modulo del login por medio de un boton podra ingresar a la pantalla de registro para ambos roles donde se autenticaran con sus necesidades dependiendo al rol que escojan. Una vez registrado le azomara una notificacion de confrimacion al correo electronico. Una vez confirmada la cuenta podra ingresar (esto aplica para ambos roles).

# Rol Administrador
Una vez hecho el proceso del registro e ingresar a la aplicacion dependiendo al rol, seran redirigidos a las necesidades plantiadas para el rol administrador en este caso, donde puede visualizar todos los registros realizados por los medidores, incluyendo fotos, coordenadas y enlaces de ubicación.

# Rol Medidor
Al igual una vez dentro de la aplicacion, podra acceder a las funcionalidades del rol medidor que puede registrar nuevas lecturas y visualizar únicamente sus propios registros de los medidodores. Ademas, de tomar fotos, obtener GPT y agregar comentarios en la aplicacion donde debera permitir losa ccesoss para poder realizar las funciones.
