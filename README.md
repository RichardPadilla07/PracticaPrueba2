# Login
El login les da autenticacion ha ambos roles (admin y medidor) pero cada uno tendra sus propias funcionalidades y necesidades. Ademas de validar sus credenciales para ver si estan registradas o correctas.

# Registro
El registro esta en el mismo modulo del login por medio de un boton podra ingresar a la pantalla de registro para ambos roles donde se autenticaran con sus necesidades dependiendo al rol que escojan. Una vez registrado le azomara una notificacion de confrimacion al correo electronico. Una vez confirmada la cuenta podra ingresar.
